- [ ] **라이선스**: 상업적 사용 가능한가?

- [ ] **버전**: 안정 버전(1.0 이상)이 있는가?
- [ ] **보안**: 알려진 취약점이 없는가? (npm audit, safety check)

- [ ] **문서**: 시작 가이드와 API 문서가 있는가?
